Formularios de consulta

Los formularios de consulta muestran un listado de todos los elementos. Desde este formulario se pueden agregar, modificar, ver los datos, exportar, filtrar, etc.



Funciones básicas de los formularios de consulta



Nuevo: Permite agregar un nuevo elemento. Nos lleva al formulario ABM que corresponda.

Modificar: Permite modificar los datos del elemento que este seleccionado en la lista.

Ver detalles: Nos muestra el mismo formulario ABM, solo que no se pueden guardar eventuales cambios que se realicen sobre los datos. Se utiliza, por ejemplo, cuando el usuario no tiene permiso para modificar.

Exportar a Word - Excel: Envía el contenido del listado a Word o Excel, para permitir la impresión, rediseño, filtrado de los datos, etc. Vale aclarar que se exporta el contenido tal como esta, o sea que si se introdujo alguna expresión para filtrar los datos, aparecerán en el listado generado solo los registros que se muestren en el listado.

Exportar a Writer – Calc: Ídem al punto anterior, pero utilizando el software de la suite OpenOffice.org.

Aceptar: Retorna el/los elemento/s seleccionado/s al formulario desde el que se llamo a la consulta. No esta siempre presente, solo aparece en caso de que el formulario deba retornar algún valor.

Cancelar / Cerrar: Cierra el formulario.
 
Filtrado
Además de las funciones descriptas, los formularios de consulta cuentan con un cuadro de texto con el cual se puede filtrar el contenido del listado.

Funciones específicas de los formularios de consulta

Aparte de las funciones ya descriptas, algunos formularios de consulta tienen otras funciones que son específicas de cada tipo de entidad. Las mismas se detallan a continuación.

Consulta de afiliados

  Registrar Pago. Esta función sirve para cuando el afiliado paga sus cuotas en el local. Se registra el pago y se anula el recibo correspondiente.

Papelera de Reciclaje

Algunos formularios cuentan con un sistema de eliminación basado en una papelera de reciclaje. Cuando el usuario elige el botón eliminar no se produce una eliminación real, sino que se envía el elemento a la papelera de reciclaje.



Eliminar el elemento de esta manera significa que el mismo no estará presente en las consultas de dicho elemento, pero si a la hora de consultar documentos anteriores donde haya participado el elemento.

Por ej: Si un afiliado es enviado a la papelera, el mismo no figurara en los listados para realizarle nuevas atenciones pero si se consulta una atención anterior de este afiliado los datos estarán visibles.

  Eliminar: Envía el elemento a la papelera.
  Papelera: Muestra la papelera. 
Restaurar: Devuelve el elemento a la consulta correspondiente.
