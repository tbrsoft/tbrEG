Activación del Software

El presente software posee dos modos de uso, el modo “demo”, que permite cierta cantidad de registros de pagos y atenciones (este modo se utiliza para probar el software sin obligación de compra), y el modo “registrado” el cual no tiene limitaciones en su uso y requiere la compra del software.

La activación del sistema (modo “registrado”) consiste en tres pasos:

1. Generar archivo de información de la PC: El sistema obtiene información única de la PC donde se instalará el software. (Menú Herramientas -> Licencia -> Generar archivo para validar equipo)
2. Enviar este archivo a tbrSoft, que analizara los datos y generara un fichero de licencia para uso exclusivo de la PC desde la cual fue generado el primer archivo.
3. Insertar el fichero de licencia. tbrSoft le enviara un archivo de licencia, que una vez incluido, activa el sistema para su uso. (Menú Herramientas -> Licencia -> Insertar Licencia)


Para comenzar a utilizar el Software de forma definitiva y no ver los datos de la base de datos de prueba, dentro de la configuracion elija la opción Base de Datos y seleccione el archivo Database.mdb en el directorio de la aplicacion. En caso de haber instalado el sistema en varios terminales y haberlos configurado mediante la configuracion automatica se recomienda copiar la base de datos de prueba a otra carpeta y renombrar el archivo Database.mdb a DatabasePruebas.mdb para evitar cambiar la configuracion en cada equipo cliente. Otra opcion es cambiar dentro de la configuracion de todos los clientes y el servidor modificando la ruta a la base de datos Database.mdb.
