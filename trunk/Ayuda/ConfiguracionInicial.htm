Configuración Inicial

Desde la version 2.0 el sistema cuenta con una pantalla de configuracion inicial con el fin de que el sistema quede listo para comenzar a funcionar de manera adecuada a su empresa u organización. A continuacion se describen cada uno de los pasos de la configuración.

1. Presentación

Es simplemente la introducción a la configuración.

 

2. Selección de la configuración de red.

Aqui debe seleccionar si el sistema va a funcionar en red o en  una sola computadora. Cuenta con dos opciones:

a. No se utilizará red, o sea, el sistema funcionará en una sola computadora.

b. Si su empresa cuenta con varios equipos y desea adquirir licencias para varios debe seleccionar la opción El sistema se utilizará en varios equipos. Luego debe elegir si la computadora donde se está instalando el sistema será la que cumplirá las funciones de servidor o será un cliente. Se recomienda que la computadora que permanecerá encendida por mas tiempo sea utilizada como servidor. Cabe aclarar que ya que el sistema cuenta con permisos por usuario, usted puede utilizar como servidor una computadora de despacho y sin embargo no habrá problemas de seguridad.

 

3. Seleccione la contraseña del Administrador.

Es recomendable que usted seleccione la contraseña en este momento. Si bien puede saltear este paso, el sistema estará expuesto hasta que asigne una clave. En caso de querer saltear este paso recuerde que la contraseña por defecto es el numero uno (1).

4. Seleccione el tipo de empresa.

A partir de su versión 2.0 el sistema cuenta con funciones tanto para Servicios de Emergencias como así tambien para Cuarteles de Bomberos. En este paso debe seleccionar el tipo de su empresa u organización. 

5. Selección del sistema de Codificación de Emergencias.

Este paso es muy importante ya que aqui se define el sistema de codigos de emergencia que se utilizará y el mismo no puede ser modificado posteriormente. Analice las opciones seleccionando en la lista y explorando los diferentes niveles en la sección inferior.

 

6. Selección de la dirección por defecto.

Aqui puede seleccionar su pais, provincia, localidad y barrio. Si bien este paso es opcional se recomienda seleccionar en este momento para que el sistema le muestre esta información en todos los campos dirección de todos los formularios donde corresponda.

 

 

Por último seleccione el botón Finalizar para que el sistema guarde la información y pueda comenzar a utilizar el software.

Es recomendado que reinicie el sistema antes de comenzar a utilizarlo.
