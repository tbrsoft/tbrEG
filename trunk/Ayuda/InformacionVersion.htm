Informació de la versión

La versión 2.0.2 de tbrEmergencyGroup cuenta con un nuevo modo de funcionamiento adecuado para los Cuateles de Bomberos.

Mejoras del software en general:

  • Se agrego un formulario de configuracion inicial, que facilita significativamente la configuracion del software especialmente en lo que respecta a la red, automatizando tareas como compartir la carpeta, configurar clientes, etc.
  • Se agrego la gestion de paises, el software se puede utilizar en cualquier pais de habla hispana.

En cuanto a las mejoras para los Servicios de Emergencias podemos mencionar:

  • Se arreglo un problema de actualizacion de los datos de las emergencias en la red. Solo se actualizaba si se refresacaba el formulario manualmente, en esta version los datos se actualizan automaticamente a traves de la red.
  • Se agrego color de fuente y negrita en las atenciones, configurable para una mejor identificacion de la prioridad de las atenciones.
  • Se simplifico el formulario de datos contables de Obras Sociales y Servicios de Emergencias.
  • Se arreglo un problema en la emision de cuotas, donde no se agregaban las areas protegidas.
  • Se arreglaron problemas menores en formularios de carga de datos.
  • Se soluciono un problema de conexion en red.

Nota
Para saber si hay nuevas versiones del Software ingresando al menu Ayuda->Acerca de... el sistema se conecta a internet y obtiene informacion de la ultima version disponible del software.
