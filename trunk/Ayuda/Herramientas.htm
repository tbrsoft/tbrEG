Herramientas

El sistema cuenta con herramientas para facilitar algunas tareas.

Importar Datos Externos

Para facilitar la carga de datos inicial, el sistema cuenta con la posibilidad de importar datos externos en diversos formatos. Puede cargar datos provenientes de un archivo de Word, Excel, cualquier documento de texto, paginas de Internet, etc. Para importar los datos debe realizar los siguientes pasos:
1. Seleccione la tabla de destino de los datos: Debe seleccionar la tabla donde se almacenaran los datos a importar. El sistema permite importar Alergias, Medicamentos, Enfermedades, Parentescos y Ocupaciones.
2. Seleccione el origen de los datos: Puede seleccionar Papelera o Archivo de Texto. Para poder importar datos desde la papelera debe seleccionar primero los datos en el archivo de origen y seleccionar Copiar. Luego seleccione la opción Siguiente.
3. Seleccione los datos a importar: El sistema le permite hacer una selección de los datos a importar para filtrar los mismos antes de ingresarlos al sistema. 

Importante
En todos los casos los datos deben estar separados un elemento por línea para que el sistema pueda reconocer los mismos. El sistema no reconoce tablas de doble entrada o de más de una columna

El sistema automáticamente ignora datos duplicados para facilitarle la tarea y evitar que el usuario deba filtrar los datos antes de importarlos.

Estado de la red

En el caso que el equipo este configurado como Servidor, esta función permite conocer los equipos que conforman la red como así también el usuario que esta utilizando cada equipo conectado. En modo Cliente, el sistema muestra el estado de conexión del equipo.

Setear último recibo

Esta función establece el último número de recibo que se haya utilizado. De esta manera el sistema numera correlativamente los recibos a partir del número establecido. Es importante recordar que se debe revisar el número de recibo cada vez que se carga una nueva serie de recibos para evitar incoherencias.

Reportes – Informes

El sistema cuenta con un sistema flexible de informes en el cual el usuario puede solicitar a tbrSoft el diseño de nuevos informes que no se encuentren incluidos en el sistema, sin necesidad de reinstalar el software. De esta manera se puede contar con informes personalizados y con la posibilidad de filtrar los resultados por diferentes criterios para lograr mayor flexibilidad.

Esta función se encuentra en el menú principal. El sistema muestra una lista de los informes disponibles de los cuales el usuario seleccionara el necesario, y mediante la opción “Mostrar listado” se podrá observar el formulario con los resultados del listado.

Algunos de los reportes pueden necesitar filtros, como por ejemplo, la fecha de la que se desea obtener la información. En este caso el sistema muestra los diferentes filtros y se le solicita al usuario que ingrese los valores correspondientes. Una vez ingresados todos los filtros se elige la opción mostrar listado para obtener los resultados, los cuales pueden ser exportados a MS Word, MS Excel u OpenOffice Calc o Write.
