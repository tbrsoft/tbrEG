Seguridad

En este capitulo se analizan los aspectos referidos a la seguridad del sistema de información.

Acceso al Sistema

Para acceder al sistema es necesario ser un empleado habilitado a tal fin y contar con una contraseña, que podrá ser modificada por el usuario y en caso de olvido, reasignada por el encargado de administración del sistema. 

Nota
La contraseña del administrador es 1 (el número uno), la cual debe ser modificada en el primer uso del sistema.


  
Si un empleado no necesita usar el sistema, como por ejemplo, un paramédico, no se le debe asignar un login cuando se da de alta el mismo. Con esto basta para que el empleado quede deshabilitado para el uso del sistema.

Una vez que se decide que un empleado puede hacer uso del sistema, se le asigna un login, que figurará a la izquierda en la pantalla inicio de sesión. El login puede coincidir con el nombre de la persona, pero esto no es necesario.      


 

Permisos

A los usuarios pueden asignárseles permisos para prácticamente todas las funciones del sistema
Al asignar permisos se debe tener en cuenta que lo importante es seleccionar los últimos nodos del árbol para un control total de las funciones que se le permiten a un usuario.

La funcion para asignar permisos se encuentra en el ABM de afiliados, en el boton Propiedades para el uso del Sistema.

 

Súper usuario

El sistema cuenta con un tipo de usuario especial llamado súper usuario. Este usuario tiene acceso a todas las funciones del sistema sin aplicársele las restricciones de permisos. Solo existe un súper usuario, denominado administrador. Es el único usuario que puede asignar permisos a otros usuarios.
