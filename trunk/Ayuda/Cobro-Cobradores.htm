Procedimiento de Cobro 1: Por cobradores

El curso normal del cobro de cuotas es de la siguiente manera:

1. Se generan las cuotas correspondientes a un determinado mes, y se imprime un listado de dichas cuotas para cada cobrador.
2. Se imprimen los recibos asociados a las cuotas generadas.
3. Los recibos son entregados a los cobradores junto al listado de cuotas.
4. Los cobradores se presentan en los hogares para realizar el cobro.
5. Los cobradores presentan el listado con las cuotas que fueron cobradas.
6. Se registran las cuotas cobradas, con lo cual se genera un resumen de rendiciones para el cobrador.

Si se presenta el afiliado para pagar en la casa central, el procedimiento es un poco diferente:
1. Se repiten los pasos 1 al 3.
2. Se presenta el afiliado, se selecciona la/s cuota/s a pagar.
3. Se registra el pago y se anulan los recibos generados para el cobrador, que son incluidos en una lista de recibos a devolver.
4. Se confecciona un nuevo recibo.


El proceso de cobro de cuotas esta reflejado en el sistema de información.

 1. Para generar las cuotas: Menú Principal -> Administración -> Emitir Listado de Cuotas a Cobrar. Seleccionar los cobradores para los cuales se desea generar las cuotas. Botón Generar Listado. En ese momento el sistema genera las cuotas, imprime los recibos y emite el listado de cuotas a cobrar para el cobrador.

2. Para registrar las cuotas que el cobrador percibió: Consulta de Empleados, seleccionar el empleado al que desean registrarse los cobros, y luego Botón “Registrar Cobro Cuotas”. Seleccionar las cuotas a cobrar de la lista, controlando el porcentaje de comisión del cobrador.		  

Si el afiliado se presenta a pagar en la casa central, el procedimiento es diferente.
1. Consulta de Afiliados -> Botón Registrar Cobro. Seleccionar las cuotas. Botón “Registrar Cobro”. El sistema registra el cobro, anula el recibo y lo incluye en la lista de Recibos anulados, e imprime el nuevo recibo.

2. El encargado de administración debe reclamar al cobrador el recibo correspondiente a la cuota que se cobro en la casa central. Una vez devueltos los recibos anulados se ingresa al formulario de registro de recibos anulados a través del menú Administración -> Registrar devolución de recibos anulados, o bien Consulta de Empleados, botón Registrar Devolución de Recibos Anulados (de esta ultima forma se muestran los recibos de este cobrador solamente). Se seleccionan los recibos que se recibieron y se presiona el botón Registrar Devolución.
